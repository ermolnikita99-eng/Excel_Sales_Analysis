# Data Analysis — Практика и учебные кейсы

Репозиторий с моими практическими работами по Data Analysis.

Здесь собраны файлы заданий, которые я выполняю по индивидуальной программе обучения (2025–2026 гг.), чтобы освоить профессию Data Analyst с нуля.

## Что находится в репозитории

- Реальные задания по анализу данных  
- Пошаговые решения в Excel / Google Sheets  
- Итоговые отчёты и дашборды  
- Файлы с расчётами, сводными таблицами, графиками, формулами и Power Query
